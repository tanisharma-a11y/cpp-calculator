# Beginner C++ Projects Portfolio
This repository contains beginner-friendly C++ projects built to demonstrate problem-solving, logical thinking, and basic programming skills. 

## Projects

### 1. Calculator
- A simple calculator that performs basic arithmetic operations: addition, subtraction, multiplication, and division.
- File: `calculator.cpp`
- Purpose: Demonstrates use of variables, user input/output, and basic logic in C++.

### 2. Carbon Footprint Calculator
- Estimates a user’s daily carbon emissions based on inputs such as travel distance and electricity usage.
- File: `carbon_footprint.cpp`
- Purpose: Combines user input and arithmetic calculations to solve a real-world environmental problem.

### 3. Programming Language Quiz Game
- A multiple-choice quiz that tests knowledge of programming languages and provides a score with feedback.
- File: `programming_quiz.cpp`
- Purpose: Shows logic implementation using conditionals and interactive user input.
